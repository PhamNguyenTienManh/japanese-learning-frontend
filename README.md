## 🎯 Giới thiệu dự án

**JAVI** là hệ thống hỗ trợ học tiếng Nhật trực tuyến, giúp người học:
- Tra cứu từ vựng, ngữ pháp, Hán tự  
- Luyện đọc, luyện thi JLPT  
- Quản lý sổ tay từ vựng  
- Theo dõi tiến độ học tập  
- Tương tác và học tập thông minh với AI
Hệ thống được xây dựng theo mô hình **Client – Server**, tách biệt Frontend và Backend.

---

## 🧱 Công nghệ sử dụng

| Thành phần | Công nghệ |
|----------|-----------|
| Frontend | ReactJS, SCSS |
| Backend | NodeJS, NestJS |
| Database | MongoDB |
| API | RESTful API |
| Xác thực | JWT |
| Công cụ | Git, Postman |

## 🚀 Hướng dẫn cài đặt

### 1️⃣ Clone source

```bash
git clone https://github.com/PhamNguyenTienManh/japanese-learning-frontend

git clone https://github.com/PhamNguyenTienManh/japanese-learning-backend
```

### 2️⃣ Cài đặt Frontend

```bash
cd japanese-learning-frontend
npm install
npm start
```

---

### 3️⃣ Cài đặt Backend

```bash
cd japanese-learning-backend
npm install
npm start
```
